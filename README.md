# Google Analytics Simulation Project (Fictional)

This project simulates a Google Analytics traffic report using dummy data, analyzed in Excel to extract insights about user acquisition and engagement.

## Tools Used
- Microsoft Excel

## Insights
- Organic Search brings the most users and sessions
- Direct traffic has the highest bounce rate
- Referral performs best in session duration
